## ⚡ TL;DR — O que é o GP?

O Grand Prix de Inovação do SENAI reúne ~26.000 alunos de todo o Brasil para desenvolver soluções para **desafios reais de empresas e indústrias**. O projeto é executado em **escuderias** (grupos de 4 a 5 pessoas).

### Fluxo resumido de desenvolvimento
```
Entender o problema → Pesquisar usuário → Idealizar solução
    → Prototipar (MVP) → Validar → Documentar no Lean Canvas
```
